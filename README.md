# Image-Stitching
Image Stitching Project

## Requirement
* numpy                 1.19.5
* opencv-contrib-python 4.4.0.46
* opencv-python         4.4.0.46
* pandas                1.1.4
* python                3.8.5



## Flow
1. feature detection
2. Feature matching
3. APAP
4. Superpixel
5. Graph construction
6. Graph cut
7. Paste target image to reference image
